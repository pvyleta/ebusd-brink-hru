Running ebusctl in HA
- use advanced terminal addon (not the standard one)
- https://github.com/hassio-addons/addon-ssh
- set password in the configuration of the addon otherwise it will not start
- "docker exec -it `docker ps | grep ebusd | awk '{print $1}'` /bin/bash"
-- access the docker of ebusd addon
-- there run "ebusctl" easily

cmd options for ebusd
- https://github.com/john30/ebusd/wiki/2.-Run#message-configuration-options

enable write params
- https://github.com/john30/ebusd/wiki/MQTT-integration#home-assistant

allow publishing everything 
- set filter to * and not "" which works only for cmd argument and not mqtt config
